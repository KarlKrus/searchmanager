Linguistics

The Linguistics module features string management treatment that manage the keyword based on the settings of each of the features.

Stored Strings

Synonyms - improves search results by allowing connecting keywords and variations with similar meaning.
Stopwords - sets the strings that should be filtered out from the external mall site search results.
Protwords - directs how certain search strings will be treated during search.

View Strings »

Did You Mean

Did You Mean module allows the user to provide list of suggestion that can be the alternative or the correct keyword to the search terms that the user also provided.

Manage Rules »

Typeahead

The Typeahead modules allows the user to configure how the search suggestion/s behave when a keyword is being typed in the search box.
