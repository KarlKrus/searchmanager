Publishing Guidelines

To push previously approved rules to production:

  • In the Show dropdown box, select Approved Rules for Publishing.
  • When the list has refreshed, tick all applicable rules.
  • Provide notes in the Comment box and click the Publish button.

Export Rule

These are all the rules that are available for Export. You can manually export rules by selecting the checkboxes and clicking on the 'Export' button.

  • If you have Auto-Export turned on, all rules that you publish to PCM will be automatically exported. There is no need to manually export any rules.
  • Exporting a rule will not remove it from the list.
