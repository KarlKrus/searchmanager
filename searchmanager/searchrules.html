Search Rules

This module allows the user to create and update rules that control the natural search results on mall sites

Elevate

The Elevate search rule is used for pushing a SKU(s) using a specific keyword to the top of the natural search result.

Manage Rules »

Demote

The Demote search rule is the opposite of Elevate search rule. By creating a Demote search rule, SKU(s) can be pushed at the bottom of search results.

Manage Rules »

Exclude

The Exclude search rule is used in removing SKU(s) from the natural search result of a specific keyword.

Manage Rules »

Redirect

Query Cleaning is another way of overriding the natural search result. If a keyword is set to have a redirection rule, the keyword will be replaced or have a redirection.

Manage Rules »

Relevancy

Creating a ranking rule is another way of manipulating natural search results. Ranking rules allow users to control how search results are displayed to a certain degree. In addition, ranking rules allow the users to control which fields are matched to a search term, and how the search results’ scores are computed.

Manage Rules »
